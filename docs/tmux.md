# :simple-tmux: tmux

## Installation

!!! quote ""

    ```shell
    sudo apt install tmux
    ```

## [Allow Notify in tmux][1]

!!! quote ""

    ```shell
    set -g allow-passthrough on
    ```

## [TPM][2]

!!! quote "Install tpm"

    ```
    mkdir -p ~/.tmux/plugins
    git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
    # Ctrl + b
    # Shift + I
    ```
