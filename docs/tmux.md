# :simple-tmux: tmux

## Installation

!!! quote ""

    ```shell
    sudo apt install tmux
    ```

## [Allow Notify in tmux][1]

!!! quote ""

    ```shell
    set -g allow-passthrough on
    ```

## TPM

Install tpm.

```
mkdir -p ~/.tmux/plugins/tpm/
git clone https://github.com/tpm/tpm.git ~/.tmux/plugins/tpm/tpm
Ctrl + b
Shift + I
```
