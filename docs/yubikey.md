# Yubikey

## [Installation][1]

```shell title="Ubuntu"
sudo apt install pcscd scdaemon
brew install ykman
```

```PowerShell title="Windows 11"
winget install Yubico.YubikeyManager
```

Add Yubico Manager program location to `PATH`.

`C:\Program Files\Yubico\YubiKey Manager`
